# Evrika (Project-Evrika)

Interactive lesson: **Archimedes' Eureka Quest** — buoyancy, density, and the crown of King Hiero.

## Quick start

```bash
npm install
npm run dev      # http://localhost:5173
npm test
npm run build
```

## Stack

- React 19 + TypeScript + Vite
- Matter.js (physics labs), Rive (landing hero), Three.js (select previews)
- Vitest + React Testing Library
