# Backend tests

There is no backend service in this repository yet. Evrika is a client-side React application (Vite + TypeScript).

When a backend is added (API, auth, persistence, etc.), place its test suites here—for example:

- `tests/backend/unit/` — pure logic, validators, serializers
- `tests/backend/integration/` — HTTP handlers, database, external services

Until then, all automated tests live under `tests/frontend/`.
